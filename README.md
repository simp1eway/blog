# Простой блог для Летнего Интенсива

## Технологии

* [Flask](http://flask.pocoo.org/)
* [Bootstrap](https://getbootstrap.com)

## Инструкция

* Установка зависимостей
```
pip install -r requirements.txt
```

* Включение DEBUG-режима
```
export FLASK_ENV=development
```

* Запуск веб-сервиса
```
flask run
```
